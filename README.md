# banti

banti can listen, see, speak and understand.

to listen banti needs following:
1. speech (what is being spoken)
2. non-speech (what sounds)
3. speaker (who is speaking)
4. emotion (how is it being spoken)

to see banti needs following:
1. understand the scene
2. recognize the faces
3. recognise the emotions on the face
4. understand the body language and gestures
5. OCR

to remeber banti needs following:
1. remember people and how are they related
2. understand and record about self
3. remeber the entire past conversations
4. in context memory

to speak banti needs following:
1. talk naturally
2. emotional expression
3. and consistent voice for everything
